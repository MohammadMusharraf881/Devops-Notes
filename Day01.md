# Day 01 – Virtualization

## What is Virtualization?

Virtualization is the technology that allows multiple operating systems to run on a single physical machine using virtual machines.

## Key Concepts

Virtual Machine (VM)
A software-based computer that runs an operating system.

Hypervisor
Software that manages and creates virtual machines.

## Types of Hypervisors

Type 1 – Bare Metal
Runs directly on hardware.

Examples:
VMware ESXi
Microsoft Hyper-V

Type 2 – Hosted
Runs on top of an operating system.

Examples:
VirtualBox
VMware Workstation
